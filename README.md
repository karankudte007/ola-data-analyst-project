# ola-data-analyst-project

To analyze ride-booking data to identify key performance drivers, understand cancellation triggers, and evaluate revenue contributions across different vehicle types.

Executive Summary: KPI cards for Total Bookings, Successful Rides, and Revenue.
Revenue Analysis: Breakdown by Payment Method (Cash, UPI, Credit/Debit Cards).
Cancellation Deep-Dive: Comparative pie charts for Customer vs. Driver cancellation reasons.
Vehicle Performance: Detailed matrix showing Distance and Value metrics for 7+ vehicle categories.
Trend Analysis: Daily booking counts to identify peak demand periods.
